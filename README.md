# 🎓 Placement Portal

A Full Stack Placement Portal developed using **Spring Boot**, **React**, and **MySQL**.

## 📌 Features

### Student
- Register
- Login
- Browse Companies
- Browse Jobs
- Apply for Jobs
- View Applications

### Company
- Add Jobs
- Manage Job Listings
- View Applicants

### Admin
- Manage Students
- Manage Companies
- Manage Jobs
- Dashboard

---

## 🛠 Tech Stack

### Frontend
- React
- Bootstrap
- Axios
- React Router

### Backend
- Spring Boot
- Spring Data JPA
- REST APIs

### Database
- MySQL

## 🚀 How to Run

### Backend

```bash
cd backend
mvn spring-boot:run
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
